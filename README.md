# 📖 Project Overview

This project predicts **health insurance costs** using **Multiple Linear Regression**.

The primary objective was to understand the complete machine learning workflow while working with a real-world dataset containing multiple numerical and categorical features.

This project builds upon my first project by introducing feature encoding, multiple input variables, and a more realistic regression problem.

---

# 🎯 Business Problem

Health insurance providers need to estimate medical insurance charges based on customer information such as age, BMI, smoking habits, medical history, and lifestyle.

Accurate cost prediction helps insurance companies make better pricing decisions while reducing financial risk.

---

# 📊 Dataset Information

- **Rows:** 15,000
- **Columns:** 13
- **Target Variable:** Insurance Charges
- **Features:** Age, BMI, Smoking Status, Diseases, Blood Pressure, Region, Lifestyle and more.

The dataset contained both numerical and categorical features along with missing values that required preprocessing.
